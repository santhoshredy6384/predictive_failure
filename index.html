Robotic Arm Health Prediction System

AI-powered predictive maintenance for industrial robotic arms

Machine Learning · Predictive Maintenance

Detect failures before they happen

A real-time health monitoring platform that uses a Random Forest classifier to predict robotic arm failure risk from live sensor telemetry — enabling proactive maintenance and minimising unplanned downtime.

Health Status Levels

Healthy — Optimal

Failure probability < 50%. All sensor readings within normal operating range.

Warning — Maintenance Needed

Failure probability 50–79%. Schedule preventive maintenance soon.

Critical — Immediate Action

Failure probability ≥ 80%. Immediate intervention required to prevent breakdown.

Key Features
🔍

Real-time Diagnostics

Select any robotic arm by ID from historical data or simulate any sensor parameter combination via the manual mode slider panel.

🌲

Random Forest Classifier

Multi-class classification (Healthy / Warning / Critical) trained on sensor telemetry with cross-validation and fit-diagnosis reporting.

📉

Confusion Matrix & Reports

Interactive heatmap confusion matrix and per-class precision, recall, and F1-score classification report rendered inside the dashboard.

⚠️

Failure Probability Score

Displays a 0–100 % failure probability gauge combining Warning and Critical class probabilities from the model's predict_proba output.

📜

Historical Data Explorer

Drill into the full sensor history for any selected robotic arm ID, sorted chronologically for trend analysis.

💾

Persisted Artifacts

Trained model and scaler are serialised with joblib so the Streamlit app loads instantly without retraining on every run.
